### JavaScript Types
  - Types in JavaScript are: number, boolean, string, undefined, null, Symbol('just me') -> ES6, objects

### Array.isArray()
  - We can use the 'isArray()' function to check if something is an array.
  ```
    var array = [1,2,3];
    
    // same as
    
    var array = {
      0: 1,
      1: 2,
      2: 3
    }
  ```
